Hybrid-Grounding Documentation
============================================

This documentation is about the Hybrid-Grounding prototype. 
If you want to know the documetation/history of Newground, or NAGG, no need to leave, you are also at the right place!
The Hybrid-Grounding prototype is the next stage in the development for a unified body-decoupled grounding prototype.

The source code can be found in the GitHub repository Newground_
